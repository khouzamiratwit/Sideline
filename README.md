
Sideline

Sideline is a sports discussion website where fans can follow games and talk about them in one place.

The site currently supports the NBA and World Cup. Users can view game information, create discussion threads, leave comments, reply to other users, and vote on posts and comments.

Built With
Next.js
TypeScript
Tailwind CSS
Prisma
PostgreSQL
Supabase
